# deltarune-en-galego-BUGS
Repo para notificar erros na tradución en Deltarune en Galego

## Atopei un erro na tradución!

- Preme en 'Issues'.
- Preme en 'New Issue'.
- Selecciona a categoría adecuada, según o erro que atopases.
- Describe brevemente o erro según o que se indica.
- Moitas grazas!


## Teño unha suxerencia ou quero mellorar a tradución

- Se é unha suxerencia menor para retocar algún diálogo, segue o proceso previo e crea un Issue coa categoría "suxerencia".
- Se estás interesado en colaborar coa tradución para implementar os teus cambios, ponte en contacto:
  - pereirocondemanuel@gmail.com
  - [@stonegebet](https://www.instagram.com/stonegebet/)
